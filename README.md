# SQL E-Commerce Sales Analysis

## Project Overview

This project analyzes e-commerce sales data using SQL to uncover customer behavior, product performance, and revenue trends.

## Business Objectives

- Identify top revenue-generating customers
- Analyze product performance
- Understand customer purchasing behavior
- Perform customer segmentation
- Calculate revenue contribution percentages

## Dataset Tables

- Customers
- Orders
- Products
- Order_Items
- Payments

## SQL Skills Used

- Joins
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Aggregations
- Ranking Functions

## Key Analysis Areas

### Revenue Analysis
- Top Customers by Revenue
- Revenue Contribution Analysis
- Running Revenue Total

### Customer Analysis
- Repeat Customers
- Customer Segmentation
- Spending Trend Analysis

### Product Analysis
- Top Products by Revenue
- Product Ranking
- Category Performance
